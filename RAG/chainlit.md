# EV PM DSS

欢迎使用 **EV PM DSS** (Electric Vehicle Product Management Decision Support System) - 电动汽车产品管理决策支持系统。

## 🎯 功能模块

### 📊 Dashboard
系统概览，快速访问最近的分析和热门竞品。

### 👥 User Insights (用户洞察)
深度分析用户画像，了解不同用户群体的关注点和痛点。

**特性**:
- 用户画像雷达图
- 痛点/爽点瀑布流
- 深度多轮问答

### ⚔️ Competitor Analysis (竞品分析)
智能竞品对比，基于真实用户评价和产品参数。

**特性**:
- 参数对比表
- SWOT 分析
- IPA 矩阵图

### 📝 PRD Writer (PRD 撰写)
AI 辅助撰写产品需求文档，支持引用溯源。

**特性**:
- 引导式需求定义
- 流式内容生成
- 引用溯源 (Ref)
- Markdown/PDF 导出

---

## 🚀 开始使用

从左侧设置中选择功能模块，然后在输入框中提出您的问题。

**示例查询**:
- "Model Y 的用户对内饰有什么评价？"
- "对比问界M5和理想L7的智驾系统"
- "帮我写一个智能座舱的需求文档"
